Healthcare · Data Engineering

Irish National Stroke Thrombectomy Registry

Designed and built the data infrastructure for Ireland's national stroke thrombectomy registry at RCSI / Beaumont Hospital. Covers REDCap database architecture, multi-role clinical workflows, data migration from three legacy systems spanning 2010–2025, and HIPE-compliant reporting dashboards.

REDCap Python SQL ETL pipelines Power BI HIPE compliance
  • Designed multi-instrument REDCap survey queue supporting registrar, nurse, interventionist, and outcomes coordinator roles
  • Built ETL pipeline harmonising categorical mismatches across three legacy datasets (Excel 2010–20, database 2020–25, REDCap 2025+)
  • Implemented HIPE-compliant field mapping and data quality validation processes
  • Developed clinical dashboards tracking procedural outcomes and door-to-reperfusion times
  • Mapped Diver BI architecture (WorkBench ETL → cBase → DivePort) for national reporting

MSc thesis combining Reddit patient sentiment with structured clinical data to assess psychiatric drug efficacy. Built a full NLP pipeline using RoBERTa, BERTopic, and VADER deployed on AWS.

RoBERTa BERTopic VADER AWS Python

  • Fine-tuned RoBERTa for domain-specific sentiment classification

  • Deployed pipeline on AWS with reproducible Docker environment
